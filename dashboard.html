How To Deploy
01
Enter client name above & download
Type the client company name and click Generate & Download. A customized .ps1 file will save to your computer.
02
Send script to client or run yourself
Transfer the script to each client machine via email, USB, or remote session. Do not post it publicly.
03
Run as Administrator on each machine
Right-click PowerShell → Run as Administrator. Navigate to the script and run it. Takes ~2 minutes per device.
04
Verify device appears in dashboard
Check the Devices view — the endpoint should appear within 60 seconds. Also verify in MeshCentral RMM.

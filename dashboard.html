How To Deploy
01
Generate from Clients view or enter name above
Click the ⬇ INSTALLER button on a client card to auto-fill their name, or type manually above. Click Generate & Download — a .cmd file saves to your computer.
02
Send the .cmd file to the client machine
Transfer via email, USB, or remote session. The file is pre-configured for this client — do not share publicly.
03
Double-click the .cmd file — that's it
Windows will prompt for Administrator access — click Yes. The installer handles everything automatically. Takes ~2 minutes per device.
04
Verify device appears in dashboard
Check the Devices view — the endpoint should appear within 60 seconds, automatically assigned to the correct client group.
